CATtoNAF
=================

Conversion from CAT XML format to NAF

1. Install module using maven
--------------------------

````shell
git clone https://github.com/alminard/CATtoNAF.git
cd CATtoNAF
mvn clean package
````

2. Usage
--------------------------

````shell
java -jar target/CATtoNAF-1.0-SNAPSHOT.jar input_file output_file
````
